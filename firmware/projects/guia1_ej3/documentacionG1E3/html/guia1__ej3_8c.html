◆ prender_apagar_titilar()

void prender_apagar_titilar ( struct leds * led_placa)

Controla el comportamiento del LED (Encendido, Apagado o Titilado). Esta función evalúa el campo estado del struct recibido y ejecuta la acción correspondiente sobre el hardware.

Parámetros
[in,out]led_placaPuntero a la estructura que contiene la configuración del LED.
Nota
Si el estado es 3 (Titilar), la función bloqueará la tarea actual debido al uso de vTaskDelay() durante el tiempo especificado en el periodo.
Atención
Asegúrese de que el puntero led_placa no sea NULL antes de invocar la función.
