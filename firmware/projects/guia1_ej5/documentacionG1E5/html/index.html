GUIA 1 - EJERCICIO 5

General Description

Escribir una función que reciba como parámetro un dígito BCD y un vector de estructuras del tipo gpioConf_t. Incluya el archivo de cabecera gpio_mcu.h Defina un vector que mapee los bits de la siguiente manera: b0 -> GPIO_20 b1 -> GPIO_21 b2 -> GPIO_22 b3 -> GPIO_23 La función deberá cambiar el estado de cada GPIO, a ‘0’ o a ‘1’, según el estado del bit correspondiente en el BCD ingresado. Ejemplo: b0 se encuentra en ‘1’, el estado de GPIO_20 debe setearse.

Hardware Connection

Peripheral ESP32
PIN_18 GPIO_20
PIN_19 GPIO_21
PIN_20 GPIO_22
PIN_21 GPIO_23
